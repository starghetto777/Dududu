# DropForge (CS2 case opening demo)

Полноценный локальный сайт с сервером и API:
- каталог кейсов
- открытие кейса с рулеткой
- баланс + пополнение
- инвентарь и история дропов

## Запуск

```bash
node server.js
```

Откройте: `http://localhost:4173`

> В этом окружении доступ к npm registry ограничен, поэтому проект сделан без внешних зависимостей.
